# Contact Book App

This is a simple contact book web app built using SvelteKit and SQLite. you can add, view, and delete contacts.

## Features
* Add a new contact (name, address, phone)
* View all saved contacts
* Delete contacts
* Data is stored in a SQLite database

## Used Tech
* SvelteKit 
* TypeScript
* better-sqlite3 (SQLite)
* Simple CSS
